# games

A small static site for hosting in-progress game playtests and collecting feedback.

## What's in this repo

- `index.md`: landing page with links to playable game builds.
- `games/`: exported HTML game builds and supporting pages.
- `games/blog/`: devlog entry index.
- `games/submit/`: feedback submission page.

## Local preview

If you are using GitHub Pages/Jekyll tooling locally, you can preview the site with your normal Jekyll workflow (for example, `bundle exec jekyll serve`) and then open the reported localhost URL.
